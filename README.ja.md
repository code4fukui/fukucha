# fukucha

fukuchaは、Unity製の2DゲームのWebGL版を提供するプロジェクトです。

## デモ
[fukucha.github.io](https://fukucha.github.io/)にてデモを公開しています。

## 機能
- ランダムガチャシステム
- キャラクターコレクション
- 簡易的なタップアクション

## 必要環境
- WebGLに対応したブラウザ

## 使い方
fukucha.github.ioにアクセスしてください。ページが読み込まれるとゲームが自動的に起動します。
